# experiment-9
Exp 3.1.1 – Secure Login Form (React):
This experiment creates a secure login form using React with hooks and form validation. It handles user input, validates credentials on the client side, and provides authentication feedback through a responsive UI.

Exp 3.1.2 – JWT Protected Routes:
This experiment implements protected routes using JWT authentication with React Router and Express middleware. It verifies tokens, restricts unauthorized access, and ensures secure navigation within the application.

Exp 3.1.3 – Role-Based Access Control (RBAC):
This experiment implements RBAC to control access based on user roles. It restricts APIs and frontend routes, provides role-based UI (like admin dashboards), and ensures only authorized users can access specific features.
